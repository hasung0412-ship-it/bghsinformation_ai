1 지도학습 (Supervised Learning)

정답이 있는 훈련 데이터를 사용하여 데이터를 학습시키는 방법입니다.

✅ 분류 (Classification)

입력 데이터가 미리 정의된 범주(그룹) 중 어느 것에 속하는지 결정하는 것을 말합니다.

😊 분류 학습에서 사용되는 용어들!

🏷️ 레이블 (Label)

인공지능이 학습할 때 각 데이터에 붙어 있는 정답 범주입니다.

예: 이 리뷰의 레이블은 '리뷰형 광고' 또는 '실제 사용 리뷰'입니다.

※ 레이블링: 데이터들에 이름을 붙이는 작업

📚 훈련 데이터 (Training Data)

인공지능을 학습시키기 위해 사용하는 데이터 모음입니다.

🧪 테스트 데이터 (Test Data)

훈련이 끝난 인공지능의 성능(정확도)을 테스트하기 위해 사용하는 새로운 데이터입니다.

2 비지도 학습 (Unsupervised Learning)
